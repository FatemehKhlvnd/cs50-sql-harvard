# DESE - Massachusetts Education Data Analysis (CS50 SQL)

This project analyzes public education data from Massachusetts using SQL.

## Dataset Overview
The database includes:
- School districts
- Schools (public and charter)
- Graduation rates
- Per-pupil expenditures
- Teacher evaluations

## Skills Demonstrated
- JOIN operations across multiple tables
- Aggregation (AVG, SUM, COUNT)
- GROUP BY and HAVING
- Subqueries
- Filtering with WHERE
- Sorting results (ORDER BY)
- Data comparison and ranking

## Key Insights
- Differences in funding across districts
- Relationship between spending and graduation rates
- Distribution of schools across cities
- Teacher evaluation performance vs funding levels
