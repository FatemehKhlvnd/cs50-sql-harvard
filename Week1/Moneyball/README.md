# Moneyball - CS50 SQL Project

This project analyzes Major League Baseball data to identify undervalued players and team performance using SQL.

## Overview
Inspired by the Moneyball strategy, this analysis explores:
- Player salaries
- Performance metrics (hits, home runs, RBIs)
- Team performance
- Cost efficiency in player recruitment

## Skills Used
- JOINs across multiple tables
- Aggregation (AVG, SUM, COUNT)
- Subqueries
- Sorting and ranking
- Filtering with conditions
- Advanced SQL analysis techniques

## Key Insights
- Salary inflation over time
- Performance vs salary efficiency
- Team-level hit analysis
- Identifying undervalued players
