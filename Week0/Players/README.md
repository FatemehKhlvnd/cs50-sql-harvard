# Players - CS50 SQL Problem Set

This project analyzes Major League Baseball player data from 1871 to 2023 using SQL queries.

## Dataset
The dataset includes player information such as:
- Name
- Birth location
- Physical attributes (height, weight)
- Batting and throwing handedness
- Career debut and final game dates

## Skills Practiced
- Filtering with WHERE
- Handling NULL values
- Logical conditions (AND / OR)
- Aggregations (COUNT, AVG)
- Sorting results (ORDER BY)
- Date filtering
- Column aliasing (AS)

## Insights
- Player demographics across countries
- Physical attribute averages over time
- Handedness combinations in baseball
- Career timelines
