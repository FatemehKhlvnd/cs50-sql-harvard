# Normals - CS50 SQL Problem Set

This project explores global ocean temperature data using climate normal measurements across different depths and geographic coordinates.

## Dataset
The dataset contains ocean temperature normals measured at different depths (0m to 5500m) across a global latitude-longitude grid.

## Key Concepts Practiced
- Filtering with WHERE
- Range queries using BETWEEN
- Aggregation functions (MIN, MAX, AVG, COUNT)
- Working with geographic coordinates
- Sorting and limiting results
- Column aliasing (AS)
- Rounding numeric outputs

## Insights
- Ocean surface temperatures vary significantly across latitude
- Equatorial regions tend to have higher average surface temperatures
- Deeper ocean layers provide additional thermal structure insights
