# Cyberchase - CS50 SQL Problem Set

This folder contains solutions for the Cyberchase SQL problem set from CS50 SQL.

## Overview
The dataset contains information about Cyberchase episodes, including season, title, topic, and air date.

## Skills practiced
- SELECT queries
- Filtering with WHERE
- Handling NULL values
- Date filtering
- Aggregation (COUNT, DISTINCT)
- Sorting results (ORDER BY)

## Files
- 1.sql → Season 1 episodes
- 2.sql → First episode of each season
- 3.sql → Production code lookup
- 4.sql → Episodes without topics
- 5.sql → Holiday episode
- 6.sql → Early released episodes
- 7.sql → Episodes about fractions
- 8.sql → Episodes count (2018–2023)
- 9.sql → Episodes count (2002–2007)
- 10.sql → Ordered episode list
- 11.sql → Season 5 titles (reverse order)
- 12.sql → Unique titles count
- 13.sql → Custom query exploration
